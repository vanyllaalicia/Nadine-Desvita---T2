## Fitur Website

* Navigasi (Navbar): Menyediakan menu untuk berpindah antar bagian halaman secara responsif.
* Halaman Utama (Hero Section): Menampilkan identitas utama berupa nama, foto, dan status pengguna.
* Tentang Saya (About): Berisi deskripsi singkat mengenai profil dan karakter pengguna.
* Pendidikan (Education): Menampilkan riwayat pendidikan secara terstruktur dalam bentuk kartu.
* Proyek (Projects): Menyajikan daftar proyek lengkap dengan gambar dan deskripsi singkat.
* Kontak (Contact): Menyediakan informasi untuk menghubungi pemilik website.
* Footer: Menampilkan identitas pembuat sebagai penutup halaman.
* Desain Responsif: Website dapat menyesuaikan tampilan pada berbagai ukuran layar.
* Teknologi: Website dibangun menggunakan HTML5 dan Bootstrap.
